{%- comment -%} Video background block - displays a full-width video background with overlaid text content. The video is embedded as an iframe (YouTube) and plays automatically without user interaction. A rich text content block is overlaid on top of the video. For YouTube videos, a thumbnail is automatically generated from the video ID (or uses a custom thumbnail_url if provided). The thumbnail is displayed until the video starts playing, then fades out via the youtube-video.js script. Parameters (via block object): - block.video_id: YouTube video ID (required) - block.video_title: Accessible title for the iframe (default: "Background video") - block.badge / block.content / block.buttons: Optional hero-style content overlaid on the video (see design-system/media-background-overlay.html) - block.class: Additional classes for the container (optional) - block.reveal: If set, adds data-reveal animation - block.thumbnail_url: URL of a thumbnail image to display while the video loads (optional, auto-generated for YouTube videos) Usage: {% include "design-system/video-background.html", block: block %} {%- endcomment -%} {%- assign _video_title = block.video_title | default: "Background video" -%} {%- assign _is_youtube = block.video_id | is_youtube_id -%} {%- if block.thumbnail_url -%} {%- assign _thumbnail = block.thumbnail_url -%} {%- elsif _is_youtube -%} {%- assign _thumbnail = block.video_id | youtube_thumbnail -%} {%- endif -%}
