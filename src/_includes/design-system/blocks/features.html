{%- comment -%} Features grid with list semantics. Parameters (via block object): - block.items: Array of feature objects with properties: - icon: Icon content. Supported formats: - Iconify ID: "prefix:name" (e.g., "hugeicons:home-01", "hugeicons:settings-02") - Image path: starts with "/" (e.g., "/images/icon.svg") - Raw content: emoji or HTML entity (e.g., "🚀") - icon_label: Accessible label for icon (optional, defaults to decorative) - title: Feature title - description: Feature description (supports markdown) - style: Optional inline styles (e.g., for custom colors) - block.reveal: If true, adds data-reveal animation (default: true) - block.center: If true, centers feature text (default: false) - block.intro_content: Optional markdown intro rendered above the grid Embeddable form: - items: pass an items array directly (skips intro_content); reveal still follows block.reveal (default true) so the embedding caller can suppress animations by passing block: { reveal: false }.
